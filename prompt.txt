# ローカル開発 - ポート変更方法

## 🔧 任意のポートで起動する設定

### 1. package.json を編集
```json
{
  "scripts": {
    "dev": "next dev -p <ポート番号>",
    "start": "next start -p <ポート番号>"
  }
}
```

**例:**
- ポート3200の場合: `"dev": "next dev -p 3200"`
- ポート4000の場合: `"dev": "next dev -p 4000"`
- ポート8080の場合: `"dev": "next dev -p 8080"`

### 2. 開発サーバー起動
```bash
npm run dev
```

### 3. アクセス
```
http://localhost:<ポート番号>
```

## 📝 ポート変更のヒント
- デフォルトは通常3000番
- 他のプロジェクトとの競合を避けるため、3001, 3040, 3200, 4000 などを使用
- ポートが既に使用中の場合は別のポート番号を試す
